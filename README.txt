¿Qué es el plugin de WordPress de Sendiroo?

El plugin de WordPress es una herramienta que le permitirá crear sus envíos en su tienda online WordPress con Woocommerce a traves de Sendiroo, de una manera sencilla, rápida y cómoda ¡en un solo click!
¿Qué necesito para usar el plugin de WordPress?

Cualquier instalación de WordPress 3.8 o superior con WooCommerce 3.0 o superior es válida para nuestro plugin
¿Cómo instalo el plugin en mi tienda?

Para instalar el plugin de WordPress siga los siguientes pasos:

    Descargue el plugin desde el directorio de plugins de WordPress
    Puede iniciar sesión en Sendiroo o crear una cuent directamente
    desde el menú de ajustes de WordPress->Opciones Sendiroo, o bien
    introducir las credenciales de API de su cuenta Sendiroo.
    Ahora nos dirigiremos en el menu de administración de WordPress
    Es necesaria una clave de APi de Google Maps para poder visualizar
    los mapas de las agencias que realizan entrega en oficina.
    Puede crear una nueva clave de API de Google Maps desde el
    enlace: https://cloud.google.com y haciendo click en 
    " Maps JavaScript API ", y creando una nueva clave de API gratuita.
    Cuando lo tenga configurado, ya puede comenzar a realizar sus envíos. 
    Para ello, diríjase a la sección Pedidos Sendiroo en el menú de administración
    de WordPress, donde podrá ver la lista de pedidos creados en WooCommerce,
    seleccione el pedido que desea enviar y dentro del detalle
    de ese pedido le aparecerá la opción de realizar el envío a través de Sendiroo.
    Puede especificar opciones adicionales como indemnizaciones o reembolsos, 
    y el precio del envío seactualizará automáticamente.

Sendiroo, your shipments at the best price

The WordPress plugin is a tool that will allow you to create your shipments in your WordPress online store with Woocommerce through www.sendiroo.es, in a simple, fast and convenient way in a single click!
What do I need to use the WordPress plugin?

Any installation of WordPress 3.8 or higher with WooCommerce 3.0 or higher is valid for our plugin
How do I install the plugin in my store?

To install the WordPress plugin follow these steps:

    Download the plugin from the WordPress plugins directory or request the plugin from your user area on our website.
    In the backoffice of your store, go to the Plugins section.
    Within Plugins, click on Upload plugin and select the .zip file that we have sent to you.
    Once the plugin has been uploaded and activated, it will appear in the list of available plugins in your online store
    Now we will go in the WordPress administration menu to Settings-> Generate Options to set the initial parameters of the plugin. 
    An APi key of Google Maps is necessary to visualize the maps of the agencies that deliver in the office. You can create a new Google Maps API key from the link: https://cloud.google.com and clicking on "Maps JavaScript API", and creating a new free API key.
    Once you have it set up, you can start making your shipments.
    To do this, go to the Generate Orders section in the WordPress administration menu, where you can see the list of
    orders created in WooCommerce, select the order you want to send and within the detail
    from that order you will see the option to send through Sendiroo.
    You can specify additional options such as compensation or reimbursements, and the shipping price will be automatically updated.
